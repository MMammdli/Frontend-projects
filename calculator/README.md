#  Calculator App

A simple and responsive **Calculator Web Application** built using **HTML, CSS, and JavaScript**.  
This project was created to practice core frontend development concepts such as DOM manipulation, event handling, and basic UI design.

## [Live Demo](https://mmammdli.github.io/Frontend-projects/calculator/)

##  Features
- Basic arithmetic operations: addition, subtraction, multiplication, division
- Clear (`C`) and delete (`⌫`) functionality
- Responsive design (works on desktop and mobile)
- Keyboard-friendly logic (optional if implemented)
- Clean and minimal UI

## Technologies Used
- **HTML5** – Structure
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Logic and interactivity


##  What I Learned
- DOM selection and manipulation
- Handling button click events
- Working with JavaScript functions and conditions
- Structuring a small frontend project
- Writing clean and readable code

##  Future Improvements

- Add scientific calculator functions
